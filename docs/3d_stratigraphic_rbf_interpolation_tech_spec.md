# [기술 명세서] 3D 지층 RBF 두께 보간 및 층서 정합성 모델링 기법
(Technical Specification: 3D Stratigraphic Interpolation using Radial Basis Functions and Cumulative Thickness)

본 문서는 GeoBIM 3D 지층 플랫폼에서 지반의 삼차원 기하학적 연속성을 묘사하고 지층 역전 및 수직 낭떠러지 왜곡 현상을 원천 방지하기 위해 탑재된 **리프로그(Leapfrog Geo) 표준 기반의 암시적 RBF 두께 보간 엔진**의 수학적 수식과 엔지니어링 아키텍처에 대해 상세히 기술합니다.

---

## 1. 개요 및 설계 목적 (Introduction & Background)

기존의 복셀 격자별 IDW(Inverse Distance Weighting, 역제곱) 가중치 투표 기반의 지층 판정법은 격자점마다 개별 독립 연산이 수행되므로 시추공 간격이 먼 공간에서 심각한 계단식 아티팩트와 수직 절벽(Vertical Cliffs) 형태의 기하학적 왜곡을 노출했습니다.

이를 극복하기 위해 본 시스템은 지하 공간을 **연속적인 수학적 경계 곡면(Implicit Curved Surfaces)**들의 집합으로 정의하고, 지질학적 퇴적 순서(Stratigraphic Sequence)에 따른 두께 단조성 제어 조건을 융합한 **방사형 기저 함수(RBF, Radial Basis Function)** 보간 기법을 도입하였습니다.

---

## 2. RBF (방사형 기저 함수) 수학적 공식화 (Mathematical Formulation)

RBF 보간법은 2차원 또는 3차원 공간상의 불규칙하게 분포된 제어점(Control Points)들로부터 전체 공간을 관통하는 가장 매끄러운 곡면을 생성하는 보간 이론입니다.

### 2.1 박판 스플라인(Thin Plate Spline, TPS) 커널
본 시스템은 가상의 금속판이 제어점들을 통과하도록 휠 때 발생하는 총 구부러짐 에너지(Bending Energy)를 최소화하는 **박판 스플라인(TPS)** 기저 함수를 사용하여 최적의 완만한 유선형 곡률을 생성합니다.
임의의 보간점 $\mathbf{x} = (lng, lat)$와 시추 제어점 $\mathbf{x}_i = (lng_i, lat_i)$ 간의 유클리드 거리 $r = \|\mathbf{x} - \mathbf{x}_i\|$에 대한 TPS RBF 커널 $\phi(r)$은 다음과 같습니다.

$$\phi(r) = r^2 \ln(r + \epsilon)$$

*(여기서 $\epsilon = 10^{-6}$은 $r=0$ 부근에서의 수치적 불안정성을 방지하기 위한 안전 파라미터입니다.)*

### 2.2 선형 시스템 구축 및 정밀 해결
보간 함수 $s(\mathbf{x})$는 방사형 기저 함수의 가중 합과 선형 다항식(평면 기울기 경향 성분)의 결합으로 구성됩니다.

$$s(\mathbf{x}) = \sum_{i=1}^{M} w_i \phi(\|\mathbf{x} - \mathbf{x}_i\|) + a_0 + a_1 x + a_2 y$$

이를 해결하기 위해 다음과 같은 대칭 행렬 시스템 $A \mathbf{w} = \mathbf{b}$를 연립하여 가중치 벡터 $\mathbf{w}$와 다항식 계수 $\mathbf{a}$를 풀어냅니다.

$$\begin{bmatrix}
\Phi + \lambda I & P \\
P^T & 0
\end{bmatrix}
\begin{bmatrix}
\mathbf{w} \\
\mathbf{a}
\end{bmatrix} =
\begin{bmatrix}
\mathbf{v} \\
\mathbf{0}
\end{bmatrix}$$

*   $\Phi_{i,j} = \phi(\|\mathbf{x}_i - \mathbf{x}_j\|)$: 제어점 간의 거리에 따른 RBF 기저 행렬
*   $P_i = [1, x_i, y_i]$: 각 제어점의 지리적 좌표 행렬
*   $\mathbf{v} = [v_1, v_2, \dots, v_M]^T$: 제어점에서의 실제 관측 데이터 값 (누적 지층 두께 또는 표고)
*   $\lambda = 0.01$: **티호노프 정규화(Tikhonov Regularization) 매개변수**로, 인접 시추공 간의 기하학적 요철이나 층서 급변에 따른 곡면 오버슈팅(Runge 현상)을 억제하고 스무딩(Smoothing) 강도를 정밀 조율합니다.

## 3. 리프로그 층서 누적 두께 모델링 (Cumulative Thickness Stratigraphic Sequence)

각 지층의 절대 표고 경계면들을 RBF로 독립적으로 피팅하게 되면, 시추 제어점이 없는 영역에서 개별 면들이 서로 교차하거나 뒤집히는(Inversion) 심각한 지질학적 모순이 발생합니다. 본 시스템은 이를 방지하기 위해 **층서 누적 두께 기반의 상속 보간 아키텍처**를 설계하였습니다.

### 3.1 절대 표고 복원 메커니즘
1.  가장 상단의 실제 지표 지형 고도를 **기준 곡면 $E_0(x, y)$**로 정의합니다.
2.  하부 지층 $k$에 대해, 지표면에서 해당 지층 하부까지의 **누적 두께 $T_k(x, y)$**를 RBF 보간 함수로 피팅합니다.
    *   $T_1(x, y)$: 토사(Soil)의 누적 두께
    *   $T_2(x, y)$: 토사 + 풍화암(Weathered Rock)의 누적 두께
    *   $T_3(x, y)$: 토사 + 풍화암 + 연암(Soft Rock)의 누적 두께
3.  피팅된 누적 두께 값을 기준으로 하위 지층의 절대 표고 경계면 $E_k(x, y)$를 하향 연산하여 복원합니다.

$$E_k(x, y) = E_0(x, y) - T_k(x, y)$$

이렇게 복원된 경계면들은 최상단 지형면 $E_0(x, y)$의 대역적 경사 및 곡률을 자연스럽게 하향 상속받으므로, 시추공 사이 공간에서 면들이 서로 침범하지 않고 평행한 흐름을 영구적으로 유지합니다.

### 3.2 핀치아웃(Pinch-out) 현상과 단조성 제어
물리적으로 지중 깊이가 깊어질수록 상단으로부터 누적된 지층의 총두께는 단조 증가해야 합니다. 따라서 보간 연산 시 아래의 **단조성 구속 조건(Monotonicity Constraint)**을 강제합니다.

$$T_k(x, y) = \max(T_k(x, y), \, T_{k-1}(x, y))$$

이 수학적 제약을 통해 특정 시추 영역에 특정 지층(예: 연암)이 아예 퇴적되지 않아 두께가 `0`인 경우, 상하부 경계 곡면이 매끄럽게 밀착하며 지층이 서서히 소멸하는 **핀치아웃(Pinch-out) 현상**을 기하학적 아티팩트 없이 유려하게 가시화합니다.

## 4. 3D 공간 분류 및 볼륨 렌더링 파이프라인

수학적 RBF 곡면들이 정의된 후, 실질적인 3D 메쉬와 복셀 블록으로 표현하기 위해 다음과 같은 공간 세분화 및 등가 곡면 추출 파이프라인을 거칩니다.

| 단계 | 프로세스명 | 설명 |
| :--- | :--- | :--- |
| **1단계** | **3D 그리드 세분화** | 관심 BBox 범위 내에서 격자 노드 $(x_i, y_j, z_l)$를 구성합니다. ($48 \times 48 \times MZ$ 격자) |
| **2단계** | **RBF 도메인 판정** | 격자의 절대 높이 $z_l$과 피팅된 층서 경계면 $E_k(x_i, y_j)$를 비교하여 지층 코드를 1~4번으로 배정합니다. |
| **3단계** | **마칭 큐브 추출** | 각 지층 코드의 도메인 볼륨 경계에서 등가 체적면을 감지하여 고품질의 삼각 폴리곤 메쉬로 변환합니다. |
| **4단계** | **복셀 픽셀 뷰 구축** | 동일한 RBF 도메인 그리드 데이터를 사용하여 인덱스 기반의 정밀 복셀(Voxel RLE) 메쉬를 평행 렌더링합니다. |

---

## 5. 엔지니어링 의의 및 실무적 기대효과

1.  **지질학적 신뢰도 극대화**: 상용 고급 지반 설계 패키지인 리프로그 표준과 동일한 층서 관계를 유지하므로 설계 실무자에게 높은 신뢰도의 3D 지각 단면을 제공합니다.
2.  **연산 속도 및 메모리 효율성**: 불필요한 고차원 공간 스무딩(3D smoothing filter) 없이도 2차원 누적 두께 변수들의 RBF 예측 조합만으로 정교한 3D 솔리드를 복원하므로 클라이언트 브라우저 렌더링 성능을 획기적으로 개선합니다.
3.  **지형 표고 잔차 완벽 연동**: 시추공의 실측 표고 고도와 AWS 지형 표고 데이터의 이격 잔차(Residuals) 보정 알고리즘과 결합하여, 실제 현장 지형 요철 위에 지층 경계면이 오차 없이 정확히 안착하도록 유기적인 연속성을 유지합니다.
