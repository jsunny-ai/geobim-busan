# Leapfrog Geo 층서 경계면 수직 하강(절벽) 현상 극복을 위한 분석 보고서

3D 지층 가시화 시 토사-풍화암, 풍화암-연암 등의 지질 경계가 완만한 유선형 경사로 흐르지 않고 마치 칼로 자른 듯 수직 절벽(Vertical Drop) 형태로 뚝 떨어지는 원인과, 이를 리프로그(Leapfrog Geo) 등 글로벌 솔루션에서 해결하는 기법에 대한 조사 보고서입니다.

---

## 1. 수직 하강(절벽) 왜곡 현상의 수학적 원인

현재 우리 시스템의 RBF 보간과 격자 분류 알고리즘에서 수직 절벽 현상이 발생하는 원인은 크게 두 가지입니다.

### ① 개별 경계면의 독립 RBF 보간 (Independent Surface Fitting)
* **현상**: 토사 하부 경계($bndElev[0]$), 풍화암 하부 경계($bndElev[1]$) 등을 각각 완전히 독립적인 RBF/TPS 곡면으로 학습시켜 예측합니다.
* **원인**: 개별 곡면의 수학적 곡률 기울기(Slope)가 완전히 따로 놉니다. 특정 전이 구간에서 풍화암 하부 TPS의 경사도가 극단적으로 가파르게 하강(Plunge)하여 수직에 가까운 면을 형성하게 됩니다.

### ② 단조성 강제 클램핑의 역전 방지 부작용
* **현상**: 지층 역전을 방지하기 위해 사용되는 아래의 클램핑이 핀치아웃 구간에서 수직 단면을 유발합니다.
  ```typescript
  if (bndElev[k] > bndElev[k - 1] - 0.05) {
    bndElev[k] = bndElev[k - 1] - 0.05
  }
  ```
* **원인**: 핀치아웃 지대에서는 두 곡면이 $0.05\text{m}$ 간격으로 완전히 포개진 채(즉, 기울기가 $100\%$ 동일하게) 나란히 흘러갑니다. 그러다가 실제 데이터가 있는 시추공 위치에 도달하면, RBF 곡면이 실제 깊은 심도(예: $-20\text{m}$)로 통과하기 위해 **급격한 낭떠러지 형태로 떨어지면서 수직 단면**이 만들어집니다.

## 2. Leapfrog Geo의 해결 기법: "기준 곡면 + 두께 보간법"

Leapfrog Geo를 비롯한 implicit 3D 지층 모델러들은 개별 곡면을 독립 보간할 때 생기는 이 수직 절벽 왜곡을 방지하기 위해 **Stratigraphic Sequence (층서학적 시퀀스)** 설계 시 **기준 곡면 및 두께 연동(Reference Surface & Thickness-based Propagation)** 기법을 채택하고 있습니다.

### ① 기준 지질 곡면(Stratigraphic Reference Mesh) 지정
* 지층 중 데이터 취득율이 가장 조밀하고 확실한 대표 경계면(예: 지표 지형면 또는 토사 하부면)을 **기준 곡면(Reference Surface)**으로 고정합니다.

### ② 두께 보간(Thickness Interpolation) 기반 전파
* 하부 경계면들을 RBF로 직접 피팅하지 않고, **[기준 곡면으로부터의 지층 실측 두께(True Stratigraphic Thickness)]를 RBF로 보간**합니다.
* 이후, **[하부 경계 = 기준 곡면 - RBF로 부드럽게 보간된 두께]** 연산을 통해 최종 경계면들을 파생시킵니다.

### ③ 유선형 경사(Pinch-out)의 수학적 구현
* 두께 값은 음수가 될 수 없으며($\ge 0$), 데이터가 없는 핀치아웃 구역을 향해 아주 자연스럽고 매끄럽게 감소하여 $0\text{m}$로 수렴합니다.
* 기준 곡면 자체가 부드러운 유선형이므로, 이 면에서 서서히 얇아지는 두께 곡면을 뺌으로써 얻어지는 하위 경계면들은 **기준 곡면의 곡률을 완벽히 물려받아 수직 낙하가 차단되고, 매우 자연스럽고 부드럽게 눕는 유선형 경사면**을 유지하게 됩니다.

## 3. 우리 프로젝트를 위한 현실적인 해결 기술 제안

현재 구조상 Leapfrog 수준의 3차원 두께 필드 보간 시스템을 통째로 재구축하는 것은 무리가 있지만, **기존의 RBF 피팅 방식을 그대로 유지하면서 수직 하강 현상을 완화하여 완만하게 눕힐 수 있는 효과적인 기법**이 있습니다.

### [방안 🛠️] TPS 스무딩 계수(Smoothing Regularization $\lambda$) 튜닝
* **원리**: 현재 TPS 보간기는 제어점을 $100\%$ 강제로 관통하는 완전 보간(Exact Interpolation)을 수행하므로, 좁은 구역에서 제어점을 맞추기 위해 곡면의 기울기가 수직으로 요동칩니다.
* **해법**: RBF 행렬 계산의 대각선 성분에 아주 미세한 **평활화 정규화 오차 계수($\lambda \approx 0.01$)**를 추가하면, 제어점을 완만하게 통과하면서 **곡면 전체의 곡률 에너지를 최소화(Smoothing)하여 수직 절벽을 아주 부드러운 경사면으로 완전히 눕힐 수 있습니다.**
